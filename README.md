# CR7

一个简单的 Vue 3 + Vite 前端项目。

## 环境要求

- Node.js 18+ (推荐 20+)
- npm 9+

## 开发

```bash
npm install      # 安装依赖
npm run dev      # 启动开发服务器 (默认 http://localhost:5173)
```

## 构建

```bash
npm run build    # 生产构建,输出到 dist/
npm run preview  # 本地预览构建产物
```

## 技术栈

- [Vue 3](https://vuejs.org/)
- [Vite](https://vite.dev/)
